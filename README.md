# AI 時代のためのバックエンド開発入門｜「動くけど分からない」を卒業する

**PHP×SQLiteで学ぶAPI・データベース・セキュリティの基礎｜AI開発の限界を乗り越える実践講座**

PHP と SQLite を使って、バックエンド開発の本質を理解する入門講座。  
AIでプロダクト開発を始めた初心者がつまずきやすい「構造理解」「設計」「データ管理」にフォーカスし、単なる実装ではなく「なぜバックエンドが必要なのか」を理解できるようにする。  

シンプルなSQLiteを採用することで環境構築のハードルを下げつつ、API・データベース・フロントエンド連携までを一貫して体験し、フルスタック開発の基礎を身につける。  

## 基本コンセプト

簡易的なアドレス帳アプリを例に、単一テーブルCRUDで「バックエンドの本質」を学ぶ

* 複雑な設計はしない
* ただし「拡張できる構造」にしておく

---

## ゴール

受講後、以下ができる状態を目指す：

* バックエンドの役割（API・データ処理・データ保存）を説明できる
* PHPで簡単なAPIを実装できる
* SQLiteを使ってデータの保存・取得・更新・削除ができる
* フロントエンドからAPIを呼び出し、アプリとして動作させられる
* AIを使ってコード生成しつつ、その内容を理解・修正できる
* SQLiteの限界と、次に学ぶべき技術（MySQLやフレームワーク）を理解している
